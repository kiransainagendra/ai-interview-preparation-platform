# InterviewAce

AI-Powered Interview Preparation Platform built using Django.

## Features

* User Registration & Login
* Resume Upload (PDF)
* PDF Text Extraction using PyPDF2
* Resume Analysis Engine
* Dynamic Skill Detection
* Resume Scoring
* Technical Interview Questions
* HR Interview Questions
* Project-Based Questions
* Resume History Dashboard
* Responsive Modern UI

## Tech Stack

* Python
* Django
* Bootstrap 5
* SQLite (Development)
* PyPDF2
* Git & GitHub

## How It Works

1. User uploads a resume PDF.
2. System extracts resume content.
3. Resume analysis engine detects technical skills.
4. Resume score is generated.
5. Missing skills are identified.
6. Personalized interview questions are generated.
7. Analysis is stored and visible in dashboard history.
